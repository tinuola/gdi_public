Responsiveness in Web Browsers

We can check the responsiveness of our web pages in at least three ways:

  • Manually resizing our browser windows
  • Physically verifying responsiveness on actual devices
  • Using the responsive viewport mode in our browsers' Developer Tools panel

Responsive Viewport Mode

Toggle the device icon in the browser's inspector window


*Open the inspector with the keyboard shortcut:
cmd opt c (Mac) or ctrl opt c (PC)

Responsive Design Design
with CSS

We can make our pages responsive using three tools in CSS:

  • Media Queries
  • **CSS Flexbox
  • **CSS Grid

**CSS Flexbox and CSS Grid are topics in Web Essentials 2 and beyond

Media Queries

  • Media queries tell the browser at what breakpoint or screen size at which to apply a specific style
  • Media queries are applied in the CSS
@media screen and (min-width: 600px){
  h2 {
    font-size: 30px;
  }

  p {
    font-size: 14px;
  }
}

A Beginner's Guide to Media Queries

Basic Media Query Syntax

A media query contains:

  • The @media keyword
  • Optionally, a media type followed by 'and'
  • The condition under which to apply the media query
  • The selector(s) to target and the CSS rule to run
@media screen and (min-width: 600px){

  h2 {
    font-size: 30px;
  }

  p {
    font-size: 14px;
  }
}
